# Jun-s-Todo-List
Basic JS Todo List

My first project intended to showcase a basic todo list app using HTML, CSS and JavaScript.
The application runs by storing tasks in local storage and will be saved unless the user clicks delete (Trash icon).

FUNCTIONS:
1. Type in task and add to list by pressing the enter key or the plus icon.
2. Pressing the check icon will mark the task complete.
3. Pressing the trash icon will remove the task from the list.
4. The drop-down menu is a filter for "completed" and "not completed" tasks, also the view "all".

No tests are available for this application.
